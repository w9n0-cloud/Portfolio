# Portfolio - Mohamed BEN HARIZ

## Pourquoi ce site ?

Ce portfolio est né d'un besoin concret : en tant qu'étudiant en **BTS SIO à l'EFREI Paris**, je voulais un espace qui me représente vraiment - pas juste une liste de compétences posée sur un CV classique, mais quelque chose de vivant qui montre ce que je sais faire en le faisant.

L'idée était simple : plutôt que de dire "je fais du développement web", le prouver directement à travers le site lui-même. Chaque ligne de code, chaque animation, chaque interaction est une démonstration concrète de mes compétences.

## La réflexion derrière

### Un portfolio, pas un template

J'ai fait le choix de tout coder **from scratch** - HTML, CSS et JavaScript vanilla, sans framework ni template. Pourquoi ? Parce que c'est la meilleure façon de montrer que je maîtrise les fondamentaux. N'importe qui peut installer un template, mais construire un site interactif de zéro, c'est autre chose.

### Structure pensée pour deux publics

Le site s'adresse à deux types de visiteurs :

- **Les recruteurs / professionnels** → ils trouvent rapidement mon parcours, mes compétences techniques et mes projets via le CV et le tableau de synthèse BTS SIO
- **Les curieux / camarades** → ils découvrent qui je suis au-delà du code, à travers la section "Au-delà du code" (sport, gaming, curiosité, esprit d'équipe...)

### Le choix du design

- **Thème sombre** avec des accents violet (#6c5ce7) - un look moderne et tech sans être générique
- **Particules interactives** en arrière-plan pour donner de la vie sans distraire
- **Animations au scroll** - chaque section se révèle progressivement pour guider le regard
- **Effet glitch** sur le nom - un clin d'œil à la culture cyber/hacker qui me passionne
- **Curseur personnalisé** - un petit détail qui montre l'attention portée à l'expérience utilisateur

## Structure du site

| Page | Rôle |
|------|------|
| `index.html` | Page principale - présentation, compétences, parcours, contact |
| `projects.html` | Galerie de tous mes projets avec filtres par catégorie |
| `cv.html` | Mon CV interactif avec animations et téléchargement PDF |
| `tableau-synthese.html` | Tableau de synthèse BTS SIO (Bloc 1, SLAM, SISR) |

| Techno | Utilisation |
